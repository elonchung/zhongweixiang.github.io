工作经历

深圳市永祥药房连锁有限公司 （ 2018.7 ~ 至今 ）

技术栈：

  • Spring Cloud (Eureka、Feign、Security、Hystrix) + Spring Boot + Mybatis + Mybatis-Plus + MySQL

  • Redis + Elasticsearch+ Kibana + Logstash + Canal + RocketMQ + RabbitMQ + Vue + Element UI

项目名称： 万药大健康商城 （ 2018.07 ~ 2019.08 ）

项目描述：

项目经验：

  • 商城以Spring Cloud+Spring Boot+Mybatis搭建的微服务架构，解决了多个模块耦合到一个应用里的弊端，更灵活的管理每个模块。
  • 使用Docker安装Elasticsearch做为商品搜索引擎，提高整体检索效率。
  • 时效不高的列表数据都使用Redis缓存，大大的提高应用的响应速度。并使用Canal抓取数据库的binlog变动数据同步到Redis里，从而保证缓存与数据库一致性。
  • 打通微信的JSAPI，H5，Native，支付宝等支付功能接口。
  • 使用Spring Security+Shiro+JWT作为登录认证授权架构。
  • 使用AOP切面技术记录接口消耗时间，统一记录错误日志。
  • 熟悉商城所有模块的业务流程，很好的应对不同的业务需求。
  • 应用程序打包及部署到测试、线上服务器。

项目名称： 医药ERP （ 2019.08 ~ 2020.01 ）

项目描述：

用于管理永祥药房线上、线下的所有资源的一套企业级管理系统，主要功能包括入库管理、出库管理、库存管理、商品管理，客户管理，财务管理等。

项目经验：

  • 4个月内完成主流程的基础开发，主要负责商品管理、采购入库、财务管理等模块业务需求开发。
  • 有效地对业务需求把控项目进度：进行评审、细化功能点，功能实现，测试，上线，回归。
  • 精确细化功能点：细化金额的变动路线，针对数据流合理设计数据表，不同操作所影响的数据流输出文档，逐个字段核对，确保准确率。
  • 引入Mybatis-Plus快速构建出基础数据接口。

项目名称： 生鲜供应链 （ 2020.03 ~ 至今 ）

用万药大健康和ERP两个项目为基础框架，搭建出来的一套生鲜供应链系统。生鲜供应是初始设计是面向工厂和政府部门饭堂做的一套供应系统。

项目经验：

  • 5个月内完成主流程的基础开发，主要负责商品管理、采购入库、财务管理等模块业务需求开发。
  • 有效地对业务需求把控项目进度：进行评审、细化功能点，功能实现，测试，上线，回归。
  • 精确细化功能点：细化金额的变动路线，针对数据流合理设计数据表，不同操作所影响的数据流输出文档，逐个字段核对，确保准确率。
  • 引入Mybatis-Plus快速构建出基础数据接口。

工作收获：

  • 以为PHP转Java比较难，因为语言的共通性，自学两个星期就开始开发增删改查业务。
  • 两年的Java开发，熟悉Java语法，多线程、集合等基础框架。
  • 熟练使用Spring家族框架，Spring Cloud分布式框架。
  • 对微服务架构的服务注册、治理问题的解决方案有一定地了解。
  • 有很强的业务解读能力，可以从容地应对商城与ERP的各种业务需求。

深圳市彬讯科技有限公司（土巴兔） （ 2016.05 ~ 2018.06 ）

项目名称： 工长项目及结算系统

项目描述：

  • 该项目是一个确保业主装修质量的一套管理系统，主要对装修工长在不同工期上的监督、业主交款、工长结算等一系列流程化管理。

项目经验：

  • 作为项目负责人，核心程序员，监督及推进项目进度，由0到1完成项目的开发；
  • 以YII框架为核心框架，设计框架路由、公用函数，全局的配置，接口规范制定，核心代码，项目重构等工作；期间参与项目需求分析与设计评审，推动各部门联调测试工作；
  • 开发后台用户权限管理、业主下单、设计师量房、工长装修工期（水电、泥木、油漆、竣工）管理；
  • 为安卓、IOS提供工长APP、设计师APP、质检APP提供数据接口。
  • 在两年里，该系统共处理全国500万装修项目，考虑项目量持续增长，PHP后台架构需升级Java后台架构，期间输出所有功能接口文档，对接Java。
  • 协助新人快速掌握开发流程，对组内人员的Code Review。

其他项目&&工作

  • 运用代码安全方面知识（CRSF、SQL注入、XSS攻击等）解决业主用户中心系统的安全问题。
  • 配合运维中心处理全站的SQL慢查询的检测与优化，从最高30秒优化到基本300毫秒以内。
  • 在一周内重构装修公司网站上线，期间加入Sphinx搜索引擎技术对装修公司列表查询进行优化处理。
  • 尝试采用sonar工具来扫描代码并进行分析。
  • 尝试接口参数通过schema统一约束，来减少数据请求出错。

项目描述：

该项目是一个自主销售的B2C商城网站和一个聚合大量家电商家线上销售的B2B平台网站，它集合了商品管理，店铺管理，商品促销，订单管理，权限管理，评论管理等一系列功能模块。

项目经验：

  • 主要负责后台开发，以Zend Framework作为基础框架，设计框架路由、公用函数，全局的配置、扩展；
  • 主要负责开发B2B平台功能有商品、店铺、订单、购物车、支付宝接口、后台的管理等；
  • 灵活应用DIV、CSS、jQuery、Ajax、Bootstrap开发前端界面效果；
  • 根据业务流合理设计数据库表结构
  • 运用Magento二次开发B2C商城，主要模块有订单，商品信息，购物车等功能模块；
  • 对线上Linux服务器安全管理和数据库维护、备份，并内网搭建测试环境、线上环境。

佛山市四维空间有限公司 （ 2013.10 ~ 2014.11 ）

项目名称：服饰B2C平台项目

该项目是一家国内销售服装DIY公司的官方网站，它集合了商品管理，商品促销，订单管理，权限管理，评论管理等一系列功能模块。本系统涵盖功能全面，用户具有操作简单，易上手，管理安全等特点，为企业的运用发挥积极作用。

项目经验：

  • 主要负责后台开发，采用smarty模板引擎搭建，PHP+MySQL技术编写，实现后台管理功能；
