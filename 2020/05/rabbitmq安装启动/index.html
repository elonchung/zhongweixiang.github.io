RabbitMQ

一、下载rlang和rabbitmq

wget https://github.com/rabbitmq/erlang-rpm/releases/download/v22.3.3/erlang-22.3.3-1.el7.x86_64.rpm

wget https://github.com/rabbitmq/rabbitmq-server/releases/download/v3.7.25/rabbitmq-server-3.7.25-1.el7.noarch.rpm

本站rabbitmq以3.7.25版本为例

二、 安装Erlang

Erlang是一种通用的面向并发的编程语言，目的是创造一种可以应对大规模并发活动的编程语言和运行环境。

rpm -ivh erlang-22.3.3-1.el7.x86_64.rpm

三、 安装RabbitMQ

rpm -ivh rabbitmq-server-3.7.25-1.el7.noarch.rpm

四、 启动RabbitMQ

cd /usr/lib/rabbitmq/bin后台启动：./rabbitmq-server start -detached

五、 关闭RabbitMQ：

./rabbitmqctl stop

六、 RabbitMQ状态：

./rabbitmqctl status

七、 开启后台管理页面

./rabbitmq-plugins enable rabbitmq_management

后台管理访问地址：http://localhost:15672/

默认账号密码都是：guest

八、 添加用户

./rabbitmqctl add_user 账号 密码

./rabbitmqctl add_user elon 123456

九、 删除用户

./rabbitmqctl delete_user username

十、 修改密码

./rabbimqctl change_password username newpassword

十一、 分配用户标签

customer为要赋予administrator权限的刚创建的那个账号的名字，Tag可以为 administrator,monitoring,management

./rabbitmqctl set_user_tags elon administrator

十二、 设置权限

即开启远程访问

(如果需要远程连接,例如java项目中需要调用mq,则一定要配置,否则无法连接到mq,admin为要赋予远程访问权限的刚创建的那个账号的名字,必须运行着rabbitmq此命令才能执行)

./rabbitmqctl set_permissions -p “/“ elon “.“ “.“ “.*”

十三、 设置开机自启

chkconfig rabbitmq-server on
